# 腹痛管理App

## 概要

カレンダーに日付単位で、その日の腹痛の状況を端末に入力・保存できる

### 保存できるデータ

- 痛み
- 原因
- メモ

### 機能一覧

- カレンダーに選択した日のイベントを追加
- 編集
- データの端末保存
- カレンダの全イベントを削除
